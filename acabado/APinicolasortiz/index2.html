Techport

Bienvenido a TechPort, el recurso de la NASA para recopilar y compartir información sobre el desarrollo tecnológico financiado por la NASA. Techport permite al público descubrir las tecnologías en las que la NASA trabaja todos los días para explorar el espacio, comprender el universo y mejorar la aeronáutica. La NASA está desarrollando tecnologías en áreas como propulsión, nanotecnología, robótica y salud humana. La NASA se compromete a hacer que sus datos estén disponibles y legibles por máquinas a través de una interfaz de programación de aplicaciones (API) para brindar un mejor servicio a sus comunidades de usuarios. Como tal, el sistema TechPort de la NASA proporciona una API de servicios web RESTful para que los datos del proyecto de tecnología estén disponibles en un formato legible por máquina. Esta API se puede utilizar para exportar datos de TechPort a un formato XML o JSON, que luego se pueden procesar y analizar.
